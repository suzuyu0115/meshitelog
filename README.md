## サービス概要

LINE通知を通じて好きな飯画像を好きな時間に飯テロできる、

ユーザー投稿型の飯テロサービスです。

## メインのターゲットユーザー
- 美味しいお店や料理を知りたい、シェアしたい人
- 食に飢えている人
- 飯テロしたい人、されたい人
- 悪戯心を持っている人

## ユーザーの抱える課題
- 美味しい料理やお店を知りたいが、自発的に調べるのはめんどくさい
- 自身の食べて美味しかった料理を他人や知り合いにシェアしたい
- 太りたいけど、食欲がなく太れない

## 解決方法
- LINEに飯テロされるので、受動的に美味しい料理、お店を知ることができる
- ランダムな日やお腹の空く深夜帯に飯テロが届き、食欲が促進される
- ユーザーを指定して送ることができるので、交流を深めることができ、飯テロしたいユーザーの悪戯心が満たされる。


## 実装予定の機能

**ログイン機能**
- LINEログイン。LIFFを使用しているため、LIFFブラウザ上で操作ができます。


**飯テロ機能**
- 飯テロをすることができます。
  - 飯テロは送り相手を選び、選んだ相手に飯テロすることができます。
    - 送り相手は、自由に選択ができる他、ユーザー全員に向けて、またはランダムなユーザー10人に送ることもできます。
  - 飯テロは時間指定投稿ができます。送り相手には指定した時間にLINE通知として届く仕様となっております。
  - 飯テロは編集及び削除することができます。

**飯受け取り機能**
- ユーザーは送られた飯テロを画像付きのLINE通知として受け取ることができます。
- 送られた飯テロはLINEのトーク画面及びアプリ上で確認することができます。

**いいね機能**
- 飯テロをいいねすることができます。

**飯一覧機能**
- 飯テロは一覧で確認することができます。
  - 一覧画面は新着順、人気順でソートできます。
- 飯名、タグから飯テロの検索ができます。
- 飯テロはX(旧Twitter)でシェアすることができます。
  飯画像がOGP画像として表示されるため、X上でも視覚的に飯テロすることが可能です。

**レコメンド機能**
- ユーザーのいいね傾向に応じて、おすすめの投稿を表示できます。


## なぜこのサービスを作りたいのか？
　私は一般男性と比較すると痩せ型体型であり、もっと太りたいという願望を持っています。

しかし、なかなか食欲が湧かず、太りたくても太れないという状況にあったため、そこで、食欲を促進してくれるサービスを作りたいと考えました。

その際に、「飯テロ」というコンテンツに着目し、通知媒体として適しているLINEを通じて避けられない飯テロを実現する案が浮かびました。

このアプリを通じて私と同じような悩みを持つ人々が少しでも減り、

体重増加の手助けとなりましたら幸いです。
